The Prairie Ridge project focused on installing a high-capacity main drainage line to serve as the backbone for an existing pattern tile system that wasn't performing due to inadequate outlet capacity.

## The Challenge

The Peterson operation had pattern tile installed across 480 acres, but the original 6" main line was undersized for the drainage area. During heavy rain events, water would back up in the system, reducing effectiveness and causing wet spots to persist.

## Our Solution

We installed a new 12" main line running 1.2 miles from the field to an improved outlet structure:

- **Main Line:** 12" corrugated pipe with proper grade
- **Junctions:** Six connection points for existing laterals  
- **Outlet:** Rock-lined outlet with energy dissipation
- **Access:** Three cleanout risers for system maintenance

## Installation Results

The project was completed in 5 working days with minimal disruption to farming operations. The improved capacity has eliminated backup issues and restored full function to the existing tile network.

Key outcomes:
- **85 acres** of wet ground now consistently productive
- **Zero backup events** during heavy rain in 2024
- **Improved system longevity** through proper maintenance access
- **Enhanced property value** with modern drainage infrastructure
